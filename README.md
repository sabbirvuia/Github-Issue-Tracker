### 3️⃣ What is the difference between map(), filter(), and forEach()?
- ans: map() use to do some thing all element of an array an create a new fuction by each elements and return it.
filter() use to check all element of a array and find the elements by conditon then create a new array of the conditional elemnets
forEach() use to loop through an array and do something for each element.
Does not return a new array, just executes a function on each item.
